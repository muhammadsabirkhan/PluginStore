PlugIn Store is a robust, full-stack e-commerce web application built with **Laravel 12** and **Tailwind CSS**. It provides a seamless shopping experience with high-end features like live AJAX search, GSAP animations, guest checkout, and a comprehensive admin management system.

## ✨ Key Features

### 🛒 Customer Experience
* **Live AJAX Search:** Real-time, two-way synced global search with instant dropdown results.
* **Dynamic Shopping Cart:** Session-based cart management allowing users to add/remove items without logging in.
* **Promo Codes & Discounts:** Apply real-time discount coupons (e.g., percentage or fixed amount) at checkout.
* **Guest Checkout:** Frictionless checkout process supporting both logged-in users and guests.
* **Product Reviews & Ratings:** Authenticated users can leave 1-5 star ratings and detailed reviews on products.
* **Wishlist (❤️):** Save favorite products for future purchases.
* **Customer Dashboard:** Track order history, view order statuses, and manage the wishlist.
* **Modern UI/UX:** Clean, responsive design powered by Tailwind CSS and smooth GSAP scroll animations.

### 🔐 Admin Panel
* **Role-Based Access Control:** Secure admin dashboard isolated from regular users.
* **Order Management:** Track and update order statuses (Pending, Processing, Shipped, Delivered).
* **Product Management:** Full CRUD operations for products, including image uploads and stock management.
* **Category Management:** Add, edit, delete, and toggle category visibility dynamically.

---

## 🛠️ Tech Stack

* **Backend:** Laravel (PHP)
* **Frontend:** Blade Templates, Tailwind CSS, Vanilla JavaScript
* **Database:** MySQL
* **Animations:** GSAP (GreenSock Animation Platform)
